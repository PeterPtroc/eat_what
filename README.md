## 今天吃什么？

命令：

1. `/增加菜品 名称 地点 时间(中午/晚上)` — 添加菜品
2. `/吃什么 [地点]` — 随机推荐（地点支持模糊匹配）
3. `/菜单 [地点]` — 查看菜品列表
4. `/删除菜品 名称` — 删除菜品（管理员）
5. `/定时推荐 [HH:MM ...]` — 开启/更新定时推荐（管理员）
6. `/取消定时推荐` — 关闭定时推荐（管理员）
7. `/吃什么帮助` — 查看帮助

### 时间段划分
08:00 - 14:00 视为 中午；其余时间全部视为 晚上。

### 定时推荐
开启后，默认每天 11:50（中午）和 17:20（晚上）自动推送。支持自定义时间：
```
/定时推荐 12:00 17:30
```
